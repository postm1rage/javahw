# Лабораторная работа №5 — JavaFX

- **JDK:** 17  
- **JavaFX SDK:** 17

При запуске добавить аргументы VM:
--module-path "C:\path\to\javafx-sdk-17.0.17\lib" --add-modules javafx.controls,javafx.fxml

## Примечание
Файлы JavaFX SDK (`.jar` в `lib/`) не включены в репозиторий.  
Перед запуском необходимо подключить свою локальную копию JavaFX SDK.
